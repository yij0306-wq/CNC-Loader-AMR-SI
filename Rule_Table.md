# AMR 시뮬레이션 핵심 규칙 테이블 (Rule Table) - V50 기준

향후 유지보수 및 추가 개발 시 참고할 시스템 핵심 규칙을 정리한 문서입니다.

| 구분 | 대상 (Target) | 동작 규칙 및 설정 값 (Rules & Values) |
| :--- | :--- | :--- |
| **택타임 (Cycle Time)** | `M3 2ND` 모델 | **105초** (개당 가공 시간) |
| | `M3 UPPER` 모델 | **125초** (개당 가공 시간) |
| | `M3 5X` 모델 | **125초** (개당 가공 시간) |
| **라이트닝 & 시각 효과** | **AMR (무인운반차)** | 이동(주행) 중일 때 차체 주변으로 고유 색상의 강력한 네온 라이트 발생.<br>(대기 중이거나 충전 중일 때는 네온 라이트 Off) |
| | **AMR 충전 베이** | 충전 베이에 도킹하여 충전 중(`CHARGING` 상태)일 때, AMR 머리 위에 번개(⚡) 아이콘이 표시됨. |
| | **CNC (제외) / 로더 본체** | 양옆 CNC 박스의 라이트닝 효과를 **삭제**하고, **로더 본체(중앙부)**가 빛나도록 설정.<br>배출 트레이 도달(DONE)과 무관하게 설비가 가동 중(`RUNNING`, `CALLING`, `DONE`)이면 **무조건 라이트닝 효과가 유지**됨. |
| | **입출고(IN/OUT) 존** | 투입 및 배출 컨베이어 존에 AMR이 도킹 완료하면 즉시 해당 구역 박스 전체에 **강력한 초록색/보라색 네온 라이트 효과**가 점멸함.<br>AMR이 도킹을 시작하는 시점부터 화물 적재/하차 후 도킹이 해제되어 빠져나가는 시점까지 불빛이 지속됨. |
| **스크롤 애니메이션** | **모든 물류 통로** | 중앙선, 상단 통로, 우측 세로 통로, 충전 진입로 등 **물류가 흐르는 모든 점선 경로**가 주행 방향에 맞춰 스크롤됨. |
